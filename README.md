# Arduino-Mega-2560-Music-Controlled-Lights-
This repository contains contains code for making music controlled lights with an Arduino Mega 2560 and the ELEGOO complete starter kit. A sound sensor takes in audio data and performs frequency analysis on it. The frequency spectrum is then maped to the LED grid. 

There is a code to measure single frequencies. This allows you to ajust the sample frequency accordingly. 

There is also a file that shows the maximum and average number of dots. This allows for calibration.
